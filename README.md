# Drain-Sensei
https://apps.apple.com/us/app/drain-sensei/id1538107952

Developed from June 2019 - December 2019

Description:  A mobile app created to service both consumers and rooting professionals with the usage of Drain Sensei devices. During the ongoing development cycle I have picked up React Native, along with various related libraries.
Environment: AWS Lambda, AWS Cognito, React, React Native, & JavaScript.
